--------------------------------------Do Zero--------------------------------------
 - git init
Para iniciar o repositório e permiti-lo executar os comandos do gitthub.
 - git remote add origin <url_do_repositorio.git>
Para parear com o repositório que deseja mexer.
 - git branch -b '<nome_branch>'
Para criar uma nova branch e depois alternar para ela.
 - git add .
Para adicionar todos os arquivos na lista de envio para o repositorio no github page.
 - git commit -m '<mensagem_de_commit>'
Para commitar suas alterações nos arquivos e enviar uma mensagem que caracterize essas mudanças.
 - git push origin '<nome_branch_atual>'
Para finalmente poder enviar os arquivos para o gihtub page.
--------------------------------------Caso sejam apenas atualizações--------------------------------------
